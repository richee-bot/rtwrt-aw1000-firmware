# RTWRT Arcadyan AW1000 — firmware releases

Public **Release** assets for the RTWRT Android app to download.

## Do not commit large .bin files into git history

Upload firmware only via **GitHub Releases** (Attach binaries).

## App download URL format

After you create a release with tag `v3` and file name:

`RTWRT-premium-v3-arcadyan_aw1000-sysupgrade.bin`

URL will be:

```text
https://github.com/YOUR_USER/rtwrt-aw1000-firmware/releases/download/v3/RTWRT-premium-v3-arcadyan_aw1000-sysupgrade.bin
```
